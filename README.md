# Badapple2-UI

### (React + TypeScript + Vite)

User interface for [Badapple2-API](https://github.com/unmtransinfo/Badapple2-API).

## Installation

```shell
npm install
```

## Run Locally

```shell
npm run dev
```

### Front-end assets

_Build:_

```shell
npx tailwindcss -i ./src/App.css -o ./dist/App.css
```

_Watch:_

```shell
npx tailwindcss -i ./src/App.css -o ./dist/App.css --watch
```

### Update dependencies

```shell
npm update --save
```
